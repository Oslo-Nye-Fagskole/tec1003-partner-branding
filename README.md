# TEC1003 Partner Branding Assets

This repository contains official branding materials provided by industry partners
for use in the [TEC1003 First Year Industry Project](https://onf.qybele.no/structure/module/overview/212.html) course.

The purpose of this repository is to make approved partner branding easily accessible
to students while ensuring consistent and correct usage across course projects.

## Scope of Use
- TEC1003 course-related work only
- Educational, non-commercial use
- For enrolled students and teaching staff
