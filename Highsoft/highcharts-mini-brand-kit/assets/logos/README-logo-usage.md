# Logos (quick usage)

For convenience, this kit includes a few common logo files in this folder.

**Self-serve Logo Lab (recommended for all variants & formats):**
https://www.highcharts.com/logolab/

## Included here
- `highcharts-logo.svg` — preferred for web
- `highcharts-logo-light.png` — for dark backgrounds
- `highcharts-logo-dark.png` — for light backgrounds

## Quick rules
- Don’t stretch or skew the logo.
- Keep clear space around the logo.
- Prefer SVG/PDF when possible (scales without pixelation).
